Projects

Here's some fun stuff I've built in my free time:

  • deal with it demo

    • Starter project that puts "deal with it" sunglasses on a person's eyes in a given photo. Built to demonstrate how to use the Microsoft Cognitive Services Face API for facial recognition. Uses Pillow (the PIL fork) for image processing.

  • quotifai

    • Simple Flask app that returns a fun quote related to the image you give it! Uses BeautifulSoup for web scraping Goodreads.com and Clarifai's API for image recognition. Try it live!

  • chordinate

    • Command-line program that generates the notes for chords commonly found in 19th century classical music. Uses Python and Western music theory.
